# Two PC Connection (Packet Tracer Lab)

## Objective
To connect two PCs and test communication using ping.

## Configuration
- PC1 IP: 192.168.1.1
- PC2 IP: 192.168.1.2
- Subnet Mask: 255.255.255.0

## Steps
- Added two PCs
- Connected using copper cable
- Assigned IP addresses
- Tested using ping

## Result
Ping successful

## Learning
- Devices in same network can communicate directly
